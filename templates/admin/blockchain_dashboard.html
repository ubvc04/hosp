How Blockchain Security Works
1. Data Storage

Medical records are encrypted and stored in the database

2. Hash Generation

SHA-256 hash of each record is generated

3. Blockchain Storage

Hash is stored on blockchain for tamper-proof verification

Benefits
  • Immutability: Records cannot be altered without detection
  • Audit Trail: Every access is logged permanently
  • Data Integrity: Verify records haven't been tampered with
  • Decentralization: No single point of failure
Technology Stack
  • Ganache: Local Ethereum blockchain
  • Solidity: Smart contract language
  • Web3.py: Python blockchain library
  • RSA-4096: Data encryption
